Using this API is super simple!

All you need to do is send a POST request to https://url-to-embed.herokuapp.com/convert,
with the request body formatted as: { "url" : "your-url-here" }

Let's say you want to convert the youtube video at https://www.youtube.com/watch?v=dQw4w9WgXcQ

Using fetch, your request would look something like:

fetch("https://url-to-embed.herokuapp.com", {
  method: "POST,
  headers: {
    "Content-Type" : "application/json"
  }
  body: "{
    "url" : "https://www.youtube.com/watch?v=dQw4w9WgXcQ" 
  }"
})
.then(res => res.json().then(data => console.log(data)))

Using axios, your request would look something like:

axios({
  method: 'post',
  url: "https://url-to-embed.herokuapp.com/convert",
  data: {
    url: "https://www.youtube.com/watch?v=dQw4w9WgXcQ",
  }
})
.then(response => console.log(response))
